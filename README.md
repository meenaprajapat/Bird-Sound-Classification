# Bird Sound Classification

A project that identifies bird species from their sounds using machine learning. It processes audio recordings and predicts the species.

## Features
- Audio processing and feature extraction
- Predicts bird species from recordings

## Technologies
- Python
- TensorFlow / PyTorch
- Librosa
- scikit-learn

## Usage
1. Add your audio files in the `data/` folder.
2. Run the prediction script:
   ```bash
   python predict.py --file path/to/audio.wav
